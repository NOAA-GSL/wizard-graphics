## Usage

To run this example, you need a [ESRI API key (legacy)](https://developers.arcgis.com/documentation/security-and-authentication/api-key-authentication/tutorials/migrate-to-api-key-credentials/). You can either set an environment variable:

```bash
export mapToken=<ESRI_API_KEY>
```

Or set `TOKEN` directly in `main.jsx`.

To install dependencies:

```bash
npm install
```

To run:

```bash
npm run dev
```
